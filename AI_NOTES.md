# AI Notes

## LLM and Provider
- **Model**: Gemini 2.0 Flash
- **Provider**: Google AI Studio
- **Why**: 
    - **Speed**: Gemini 2.0 Flash provides fast task generation.
    - **Context Window**: Able to process large amounts of documentation or feedback.
    - **Cost**: Good price-to-performance ratio for structured data generation.

## What AI was used for
- **Core Engine**: Generating JSON structures for User Stories, Engineering Tasks, and Risk Blocks.
- **UI Scaffolding**: Drafting initial Tailwind layouts and component structures.
- **Copywriting**: Generating help text and feature descriptions.

## What was checked manually
- **Auth Flow**: Verified session handling and Google Auth redirects.
- **Database Schemas**: Tuned MongoDB document structure for history querying.
- **Performance**: Optimized Svelte 5 runes to prevent unnecessary re-renders.
